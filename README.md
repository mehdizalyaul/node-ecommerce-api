# Ecommerce Backend

This is the backend for the Ecommerce project. It is a RESTFul API built with Node.js, Express, and MySQL.

## Installation

1. Clone the repository

```bash
git clone git@github.com:mehdizalyaul/ecommerce-project.git
```

2. Install dependencies

```bash
cd backend
npm install
```

3. Copy the `.env.example` file to `.env` and update the environment variables

```bash
cp .env.example .env
```

4. Run the server

```bash
npm start
```
